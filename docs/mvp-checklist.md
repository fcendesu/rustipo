# Rustipo MVP Checklist

This checklist is derived from the acceptance criteria in `rustipo_prd.md`.

## Acceptance criteria

- [ ] `rustipo new my-site` creates a usable starter portfolio project
- [ ] `rustipo build` converts Markdown content into a static site in `dist/`
- [ ] Generated site includes homepage, about, resume, blog, and projects pages
- [ ] Blog and project entries render with frontmatter metadata
- [ ] Selected theme is loaded and applied during rendering
- [ ] Theme assets and user static assets are copied into output
- [ ] `rustipo serve` serves the built site locally
- [ ] Invalid config/theme/content produces readable errors
- [ ] Repository is publishable as an open-source project
