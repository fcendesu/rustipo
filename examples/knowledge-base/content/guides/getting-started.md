# Getting Started

This page is intentionally simple so the surrounding docs theme can do most of the structural work.

## Create a site

```bash
rustipo new my-site
cd my-site
```

## Preview locally

```bash
rustipo dev
```

## Build for production

```bash
rustipo build
```
