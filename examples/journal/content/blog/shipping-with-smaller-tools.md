# Shipping With Smaller Tools

The more knobs a site has, the easier it is to spend a day configuring the site instead of writing for it.

## The threshold

A good publishing stack should disappear once the structure is in place.

> [!TIP]
> When a project is content-first, the editing loop matters more than framework optionality.

## What stays important

- predictable routes
- fast preview builds
- templates that do not fight the content model
- output files that are easy to deploy anywhere

That is why this example keeps the homepage small and lets the writing index do the heavy lifting.
