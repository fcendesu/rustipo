# Basic Portfolio

The basic portfolio example is still the starter site generated by `rustipo new`.

## What It Shows

- local default theme structure
- homepage plus standalone pages
- blog and project sections
- starter layout and typography controls in `config.toml`

## Best Use

Use it when you want the most direct path from zero to a working personal site.
